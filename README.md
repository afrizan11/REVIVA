# Reviva ??

Reviva adalah sebuah platform inovatif berbasis web untuk pengelolaan, pelaporan, dan pengumpulan minyak jelantah dari pengguna ke pengepul demi mendukung kelestarian lingkungan.

## ?? Fitur Utama

- **Halaman Publik**: Beranda, Tentang Kami, Kontak, Tim, dan Metodologi.
- **Member / User Panel**:
  - Registrasi Akun
  - Dashboard Pengguna (\dashboard-user.html\)
  - Riwayat Penyetoran (\history-user.html\)
  - Pelaporan (\laporan-user.html\) dan Pengaturan Akun
- **Admin Panel**:
  - Dashboard Admin (\dashboard-admin.html\)
  - Manajemen Data Minyak (\minyak-admin.html\)
  - Manajemen Pengguna (\users-admin.html\)
  - Laporan Keseluruhan (\laporan-admin.html\) dan Pengaturan Sistem

## ?? Struktur Direktori Utama

- \/reviva/\ - Berisi seluruh file utama antarmuka (HTML).
- \/reviva/css/\ - Berisi file gaya atau stylesheet (\style.css\, \dashboard.css\).
- \/reviva/js/\ - Berisi interaktivitas dasar proyek (\main.js\, \dashboard.js\).
- \/reviva/img/\ - Berisi seluruh aset visual.

## ?? Cara Menjalankan Program

Karena proyek ini berbasis front-end statis mentah (HTML/CSS/JS):
1. Buka folder proyek menggunakan file manager.
2. Masuk ke dalam direktori \
eviva\.
3. Klik ganda pada \index.html\ untuk membuka di browser.
4. *Opsional / Direkomendasikan*: Buka folder melalui Visual Studio Code, gunakan ekstensi **Live Server**, klik kanan pada \index.html\ -> *Open with Live Server*.
